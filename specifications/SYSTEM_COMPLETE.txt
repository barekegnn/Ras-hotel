API ENDPOINTS IMPLEMENTED (28 total)
================================================================================

AUTH
  POST   /api/v1/auth/login           — Staff login with lockout
  POST   /api/v1/auth/logout          — Clear session

ROOMS
  GET    /api/v1/rooms                — List available rooms (public)
  GET    /api/v1/rooms/[id]           — Room details
  POST   /api/v1/rooms                — Create room (manager)
  PATCH  /api/v1/rooms/[id]           — Update room (manager)
  DELETE /api/v1/rooms/[id]           — Deactivate room (manager)

BOOKINGS
  GET    /api/v1/bookings             — List bookings (paginated, filtered)
  POST   /api/v1/bookings             — Create booking (online or walk-in)
  GET    /api/v1/bookings/[id]        — Booking details
  GET    /api/v1/bookings/lookup      — Guest self-service lookup
  POST   /api/v1/bookings/[id]/checkin — Check-in action
  POST   /api/v1/bookings/[id]/checkout — Check-out action
  POST   /api/v1/bookings/[id]/extend — Extend stay (preview + confirm)
  POST   /api/v1/bookings/[id]/no-show — Mark as no-show
  POST   /api/v1/bookings/[id]/cash-payment — Record cash payment

PAYMENTS
  POST   /api/v1/payments/chapa-init       — Initialize Chapa checkout
  POST   /api/v1/payments/chapa-webhook    — Webhook handler (verify signature)

REPORTS
  GET    /api/v1/reports/occupancy        — Occupancy analytics (manager)

SHIFT NOTES
  GET    /api/v1/shift-notes               — List shift notes (paginated)
  POST   /api/v1/shift-notes               — Create shift note
  DELETE /api/v1/shift-notes/[id]          — Delete shift note
